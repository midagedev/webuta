WebUtau Korean V3 Listening Review

Listen to the generated V3 WAVs and generate listening-scores.local.json for the release audit.

No recording step: this review does not ask anyone to record a voice. It only scores WAVs generated from the bundled synthetic UTAU V3 voicebank.

Template path: listening-scores.local.template.json. A phrase should score 4/5 or higher before community release.

After downloading this JSON, download handoff-report.local.json from the DAW handoff page too, keep both files in Downloads, then run npm run release:evidence-status before accepting both with npm run release:accept-evidence. Use explicit --scores and --handoff paths only when the files are somewhere else.

10-minute listening review path

manual evidence only after real listening
01 Set reviewer info Fill reviewer, playback device, and choose a passing decision only after every required score is honest.
02 Listen phrase by phrase Use headphones or neutral speakers, then score Korean clarity, vowel stability, consonants, musicality, and artifacts.
03 Compare V3 against V2 For each phrase, score whether current V3 is clearly better than the legacy V2 baseline.
04 Download the JSON Only download listening-scores.local.json after the status says the release scorecard passes.
phrases 4 V3 renders
comparison 4 legacy V2 WAVs
threshold 4/5 or higher
output listening-scores.local.json
