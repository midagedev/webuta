Fast Acceptance Path

The machine-readable reviewer packet is available as release-packet.json. It lists the current V3 voicebank, review WAVs, required evidence files, and final commands.

Download listening-scores.local.json and handoff-report.local.json, leave both files in your Downloads folder, then check them without copying anything:

npm run release:evidence-status

When the status report says both files are ready, run:

npm run release:accept-evidence

The command auto-detects the newest matching files from Downloads, including browser-renamed files like listening-scores.local (1).json. Use the longer command below only when the files are somewhere else.

npm run release:accept-evidence -- --scores path/to/listening-scores.local.json --handoff path/to/handoff-report.local.json

npm run release:audit-utau

Single-file compatibility commands

npm run voicebank:accept-review-v3 -- --scores path/to/listening-scores.local.json

npm run release:accept-daw-handoff -- --handoff path/to/handoff-report.local.json
